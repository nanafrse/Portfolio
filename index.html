Nanaflix: Um Recomendador de filmes

Este projeto desenvolvido com a biblioteca p5.js tem como objetivo criar um sistema interativo de recomendação de filmes com base na idade do usuário e nos seus gêneros de preferência. A interface gráfica é simples e amigável, composta por uma caixa de texto onde o usuário insere sua idade e várias caixas de seleção (checkboxes) representando diferentes gêneros de filmes, como Ação, Comédia, Drama, Animação, Terror, Romance, entre outros.

Ao preencher a idade e selecionar os gêneros que mais gosta, o usuário pode clicar em um botão que ativa o sistema de recomendação.

O programa, então, analisa as informações fornecidas: primeiro, verifica se a idade inserida é compatível com certos gêneros (por exemplo, não recomendar filmes de terror para crianças), e depois cruza os gêneros selecionados com uma base de dados simples contendo filmes previamente cadastrados por categoria e faixa etária. Com essas informações, o sistema exibe uma lista de filmes recomendados na tela.

Este projeto é ideal para iniciantes em programação criativa com p5.js, pois utiliza conceitos básicos de entrada de dados, estruturas condicionais, arrays e elementos interativos da biblioteca. Além de ser funcional, o projeto também permite diversas possibilidades de expansão, como adicionar imagens dos filmes, trailers, classificações ou até recomendações com base em múltiplos critérios, oferecendo uma boa base para projetos mais complexos no futuro.

Quente e frio; um jogo de esconde-esconde!

"Quente e Frio", é um jogo interativo desenvolvido com a biblioteca p5.js que pode ser analisado como um produto digital voltado para o entretenimento e o aprendizado de lógica básica de programação. A mecânica do jogo é baseada em um desafio de localização: o sistema escolhe aleatoriamente uma posição secreta dentro da área visível da tela, e o usuário deve mover o cursor do mouse tentando descobrir onde está esse ponto oculto.

A cada movimento do mouse, o sistema calcula a distância entre o cursor e o ponto alvo, e responde visualmente por meio de mudanças de cor no fundo da tela. Quando o usuário está longe do ponto, são exibidas cores "frias", como azul e roxo; à medida que se aproxima, as cores vão esquentando, passando por tons como amarelo, laranja e vermelho, indicando que está chegando perto. O projeto se destaca por sua simplicidade, mas também pela eficácia com que traduz lógica em feedback visual, tornando a interação intuitiva e envolvente.

Como produto, é ideal para fins educacionais ou como demonstração de como conceitos como variáveis, distâncias e condicionais podem ser aplicados de forma criativa.

Além disso, o projeto possui grande potencial de expansão: é possível adicionar elementos como sons, cronômetro, níveis de dificuldade ou desafios extras, transformando-o em um jogo mais robusto. "Quente e Frio" é um exemplo de como uma ideia simples pode ser transformada em uma experiência digital rica, interativa e divertida.

Run: um jogo de corrida entre amigos!

Este projeto de corrida, desenvolvido com a biblioteca p5.js, é a continuação de uma versão inicial e traz aprimoramentos que o tornam mais completo e interessante como produto interativo. Na parte 2 do projeto, a dinâmica básica do jogo — onde um carro controlado pelo jogador percorre uma pista enquanto desvia de obstáculos e interage com outros elementos — é expandida com recursos adicionais que aumentam o desafio e a imersão.

Entre as melhorias, destaca-se a adição de um sistema de pontuação, que contabiliza os pontos a cada vez que o jogador ultrapassa os carros adversários, incentivando o desempenho e criando um objetivo claro. Também são introduzidos elementos como colisão entre veículos, que interfere no jogo ao reduzir a pontuação ou reiniciar a posição do carro, o que adiciona uma camada de dificuldade e estratégia. Outro ponto relevante é o uso de sons e trilha sonora, que contribuem para a ambientação e tornam a experiência mais dinâmica.

A movimentação contínua da pista e dos carros adversários reforça a sensação de velocidade, enquanto o controle do carro principal por meio das teclas direcionais mantém a jogabilidade acessível. Como produto digital, esta versão do projeto de corrida se posiciona como um jogo casual, ideal para ser executado em navegadores, com potencial para expandir ainda mais com novos modos de jogo, fases, power-ups ou rankings. No aspecto técnico, ele explora conceitos importantes como movimentação em dois eixos, detecção de colisão, controle de entrada do usuário, sons e gerenciamento de estados do jogo. É um exemplo eficiente de como a programação criativa pode ser usada para construir experiências completas, combinando lógica, arte e interatividade.
